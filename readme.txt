OpenVPN Github Repository

This repository contains the configuration files for an OpenBSD 5.2 server that is a functions as a combined router/firewall/VPN server (OpenVPN 2.2.2). It accompanies a technical post:

http://www.sclaggett.com/Setting-up-a-router-firewall-VPN-combo-with-OpenBSD-5.2-and-OpenVPN-2.2.2/
